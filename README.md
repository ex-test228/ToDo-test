# 作成した経緯
pythonとデータベース操作の学習のために作成。ToDoの登録・更新・削除が可能。

# 使用技術
python 3.12
flask

# テスト実行手順
リポジトリのクローン
```
git clone https://github.com/ex-test228/ToDo-test.git
```

ディレクトリの移動
```
cd ToDo-test
```

依存関係のインストール
```
pip install -r requirements.txt
```

実行  
macOS / Linux:
```
python3 main.py
```
Windows:
```
python main.py
```
